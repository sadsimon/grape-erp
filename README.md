# 技术框架
* 后端框架：SpringBoot 3.4.2
			maven： 3.9.9

* 前端框架：Vue 3.4.38 + typescript 5.4.2,
* UI框架: element-plus 2.8.5

# 开发环境
建议开发者使用以下环境，可以避免版本带来的问题
* IDE: IntelliJ IDEA 2019.2+和HBuilder X最新版
* DB: Mysql 8
* JDK: JDK 17
* Node: Node [18.x、20.x]
* Maven: Maven 3.2.3+
* Redis: 7.4.2
* Nginx: 1.26.3
